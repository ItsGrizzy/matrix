How to Use Matrix

Quick Start

  1. Enter your target: IP address, domain, or network range
  2. Add credentials: Username and password (optional)
  3. Select tool: Choose from NetExec, Nmap, Web App, etc.
  4. Copy commands: Click any command card to copy

Features

  • Works with any input: Commands work with just IP or full credentials
  • Real-time updates: Commands update as you type
  • 20 commands per page: Fast loading with pagination
  • Export functionality: Download commands as TXT files
  • Click anywhere to copy: Entire command cards are clickable

Shortcuts

  • Ctrl + K: Focus search box
  • Escape: Close modals
